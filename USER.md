## 소개

## 사전작업
* NVIDIA GPU가 설치 된 Ubuntu 18.04 에서 작동합니다.
* Python 3.x 에서 작동합니다.
* 사전에 학습이 필요합니다.

## 설치
* (선택)virtualenv를 이용하여 python 가상환경을 만듭니다.
```
$ virtualenv -p python3 venv
$ source venv/bin/activate
```
* 필요한 패키지를 설치합니다.
```
$ pip install -r requirements.txt
```
